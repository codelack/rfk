Current Live Programming

Radio Free Kokomo offers a diverse and eclectic mix of programming that you won't hear anywhere else, with a focus on bands and artists that don't get much, if any airplay on mainstream radio.

We broadcast 24/7, and when there isn't live programming schedule, we run blocks of automated programming that features more of the great alternative and underground music you won't hear elsewhere, occasionally mixed in with familiar favorites.
